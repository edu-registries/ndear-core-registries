# ndear-core-registries
NDEAR Core Registries

## Running
1. Clone the repository  
2. Start docker compose, schema configuration from the repository are mounted on registry with this.
    ```
        docker-compose up
    ```
3. Verify the api 
    ```
    curl -v localhost:8081/health

    ....
    ```
